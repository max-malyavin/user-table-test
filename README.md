Интерфейс для управления пользователями.


## Используемые технологии

* ReactJS (useState, useEffect, useDispatch, useSelector)
* Redux `&` React-redux `&` Redux-thunk `&` Redux Toolkit
* React Router Dom `&` withRouter
* Axios
* SASS
* Ant Design UI
